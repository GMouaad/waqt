⏰ Waqt

Time Tracking Application

A portable Python time tracking application for managing work hours, overtime, vacation days, sick leaves, and work activities.

Get Started View on GitHub

Features

One-Click Timer

Interactive dashboard timer with Start, Stop, Pause, and Resume controls for real-time tracking

Work Hours Tracking

Track daily work hours (8 hours/day, 40 hours/week standard)

Overtime Calculation

Automatically calculate overtime hours

Vacation Days

Track and manage vacation days

Sick Leaves

Record sick leave days

Reports

View weekly and monthly overtime summaries

CLI Tool

Command-line interface for quick time tracking from the terminal

MCP Server

Model Context Protocol server for AI assistant integration

Installation

Quick Install (Recommended)

One-line installer for your platform:

curl -fsSL https://raw.githubusercontent.com/GMouaad/waqt/main/install.sh | bash
irm https://raw.githubusercontent.com/GMouaad/waqt/main/install.ps1 | iex

Using uv (from source)

git clone https://github.com/GMouaad/waqt.git
cd waqt
uv venv
source .venv/bin/activate  # On Windows: .venv\Scripts\activate
uv pip install -e .

After installation:

  • waqt --version - Check version
  • waqtracker - Start the web server (http://localhost:5555)
  • waqt start - Start time tracking from CLI
  • waqt summary - View summary

Usage

Web Interface

Navigate to http://localhost:5555 after running waqtracker

  • Dashboard Timer: One-click start/stop/pause/resume
  • Time Entry Management: Add, view, and delete entries
  • Reports: Weekly and monthly summaries
  • Leave Management: Track vacation and sick days

CLI Commands

# Start tracking time
waqt start --time 09:00 --description "Working on feature X"

# End tracking time
waqt end --time 17:30

# View summary
waqt summary

# Export time entries
waqt export --period week

# Configuration management
waqt config list
waqt config set weekly_hours 40
