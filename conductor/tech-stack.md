# Technology Stack

## Core Technologies
- **Python 3.11+:** The primary programming language, chosen for its robustness, rich ecosystem, and excellent support for time-handling and web development.
- **Flask:** A lightweight and flexible WSGI web application framework, ideal for building a self-contained, portable application.
- **SQLite3:** A self-contained, high-reliability, full-featured, SQL database engine. It is the perfect choice for local data storage without external dependencies.
- **SQLAlchemy:** The Python SQL Toolkit and Object-Relational Mapper that gives the application the full power and flexibility of SQL while maintaining Pythonic code structures.

## Frontend Stack
- **Modern HTML5 & CSS3:** Utilizing semantic structures and advanced CSS features like `backdrop-filter` and Flexbox/Grid for a modern, responsive interface.
- **JavaScript (Vanilla):** Leveraging modern ES6+ features for interactive elements and dynamic updates without the overhead of heavy frontend frameworks.
- **Glassmorphism Design:** A specialized CSS-driven aesthetic emphasizing transparency, blur, and depth to deliver a professional and contemporary user experience.

## Tooling & Infrastructure
- **uv:** An extremely fast Python package and project manager, used for dependency management and environment isolation.
- **Pyinstaller:** Utilized to bundle the Python application into standalone executables for various platforms, ensuring maximum portability.
- **Pytest:** A mature and feature-rich testing framework used to ensure code reliability and maintain high test coverage.

## Project Architecture
- **Monolithic Portable App:** A unified codebase designed to run locally, prioritizing ease of installation and data privacy.
- **CLI & Web Dual Interface:** Providing users with both a powerful command-line tool and an intuitive web dashboard.
- **Model Context Protocol (MCP) Integration:** Exposing application features to AI assistants for enhanced productivity.
